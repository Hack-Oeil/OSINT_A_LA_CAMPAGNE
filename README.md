# CTF OSINT - Pain de campagne

## Présentation du CTF 
**ID** 39 dans **les CTFs de Cyrhades**


# Format du Flag
Latitude au format : xx.xxx
Longitude au format : y.yyy
séparé par une virgule, comme ceci : xx.xxx,y.yyy
